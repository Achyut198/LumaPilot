# LumaPilot v4.3.3

Latest stable update with display-resolution controls and reset improvements.

## Highlights
- Added per-display **Resolution** submenu in menu bar.
- Added **Reset to Default** action for resolution.
- Marked default resolution with a **Default** label.
- Global **Reset Settings** now also resets display resolutions to defaults.
- Per-display reset in Display preferences also resets that display's resolution.
- Added latest-install artifacts in `dist/`:
  - `LumaPilot-latest-macOS.dmg`
  - `LumaPilot-latest-macOS.dmg.sha256`

## Installable Artifacts
- `dist/LumaPilot-v4.3.3-macOS.dmg`
- `dist/LumaPilot-v4.3.3-macOS.dmg.sha256`
