# LumaPilot v4.3.9

- Adds Option+number display toggle shortcuts for quickly turning connected and known displays off/on.
- Keeps shortcut display ordering stable when a monitor is disabled, so pressing the same shortcut toggles the same display back on.
- Makes the menu bar icon theme-adaptive by using the system template rendering color.
- Publishes an unsigned/ad-hoc release fallback when notarization credentials are unavailable, keeping the README command-line install path working.
