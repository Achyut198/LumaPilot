# LumaPilot v0.1.0

First public release of LumaPilot for macOS.

## Highlights
- Per-display controls in menu bar:
  - Brightness slider
  - Volume slider
  - Display toggle (disconnect/reconnect)
- Built-in display support improvements:
  - Built-in display volume control shown when system output volume is controllable
- Safety improvements:
  - Keep at least one display enabled protection
  - Disconnect warning dialog with cleaner compact UI
- Keyboard shortcuts:
  - Brightness / Contrast / Volume / Mute
  - Display Off and Display On
- Settings updates:
  - Working Auto Start (Start at login) option (macOS 13+)
  - Keyboard settings include display on/off shortcut recorders
